Comandos do Termux

[OBS]: Esses comandos devem ser excutados apenas 1 vez.
  • /$ apt upgrade
  • /$ apt update
  • /$ apt install git -y
  • /$ apt install nodejs -y
  • /$ apt install ffmpeg -y
  • /$ apt install python3 -y
  • /$ pip install flask
[OBS]: Comandos apenas para verificar as instalaçoes
  • /$ git --version
  • /$ node --version
  • /$ python --version
[OBS]: Clonar o repositório do Bot
  • /$ cd /sdcard
  • /$ git clone https://github.com/Kamado8421/mdevbot-md-update.git
[OBS]: Para rodar o Bot (sempre que o termux for fechado)
  • /$ cd /sdcard
  • /$ cd mdevbot-md-update
  • /$ npm start
